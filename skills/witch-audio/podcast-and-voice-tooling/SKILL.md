# Podcast and Voice Tooling

Use this skill when building or evaluating tools for spoken-word audio.

## Core priorities

For voice, clarity beats spectacle.

## Common workflow pieces

- speech cleanup
- silence/segment detection
- transcript alignment
- chaptering/highlight extraction
- loudness normalization
- noise reduction
- speaker separation/tagging

## Common mistakes

- over-denoising until speech sounds brittle
- too much compression with no context
- no transcript-aware workflow
- no chapter/segment structure for long content

## Output style

Prefer:
- clarity goal
- workflow stages
- which steps can be automated
- what still needs human review

Voice tools should save time without killing humanity.
