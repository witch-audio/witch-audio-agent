# Suno / Udio Prompting

Use this skill when someone wants better prompts for AI music tools or wants to turn a production idea into a generative-music workflow.

## Prompt rule

A good prompt balances four things:
- genre anchor
- mood/emotion
- instrumentation/texture
- structure or energy arc

Too vague and it drifts.
Too specific and it can get stiff.

## Good prompt ingredients

Reach for:
- 1–2 genre anchors
- 3–5 mood/texture words
- specific instrument cues
- vocal/no-vocal instruction
- production adjectives
- optional era or scene reference

## Prompt template

[genre / subgenre], [mood], [instrumentation], [vocal style], [production adjectives], [energy arc]

Example:
melancholic ambient synthpop, soft female vocals, washed pads, pulsing bass, warm analog texture, spacious reverb, intimate verse that blooms into a bigger luminous chorus

## What usually works

- clear instrumentation
- emotional contrast
- strong texture words
- energy movement across the song
- concise reference framing without overstuffing

## What usually fails

- giant comma soup
- too many genre labels fighting each other
- only describing lyrics theme with no sound cues
- asking for hyper-technical mix language with no musical direction

## Prompt dimensions to control

### Genre
Use one primary and maybe one secondary genre.
Example: indie electronic + dream pop

### Mood
Pick a lane.
Examples: haunted, euphoric, intimate, weightless, feral, nostalgic

### Instrumentation
Say what should actually be heard.
Examples: detuned saw pad, plucky mallets, broken drum machine, clean strat, granular vocal textures

### Vocals
State:
- male / female / duet / no vocals
- breathy / intimate / aggressive / choral / robotic
- foreground vs buried in mix

### Production feel
Examples:
- tape-worn
- glossy
- lo-fi
- cinematic
- club-ready
- overdriven
- dreamy and diffuse

## Iteration workflow

1. Start with one clean prompt.
2. Listen for what is missing.
3. Add only 1–2 stronger cues on the next pass.
4. If drift is bad, simplify instead of adding more clutter.
5. Save successful phrases as reusable style blocks.

## Bridging from music language

Translate user input like this:
- “make it hit harder” -> stronger drums, more transient presence, bigger chorus lift
- “more dreamy” -> softer transients, wider pads, diffuse reverb, hazier top end
- “less cheesy” -> remove generic epic language, reduce overblown adjectives, tighten genre anchor
- “more human” -> intimate vocals, imperfect phrasing, restrained production, less synthetic descriptors

## Useful prompt chunks

### Dreamy
hazy, luminous, washed-out pads, soft-focus synths, floating reverb, intimate and nostalgic

### Dark
brooding low end, cold textures, sparse percussion, ominous synth layers, shadowy atmosphere

### Energetic
driving drums, rising tension, bright hooks, punchy bass, momentum that keeps building

## Output style

When helping, prefer:
- 1 strong prompt
- 2 alternate prompt directions
- one sentence on why each differs

Keep prompts musical, not bloated.
