# DAW Music Production

Use this skill when the user wants witch.audio to help make music inside a DAW, not just talk about music theory.

## Core role

Act like a sharp producer/technical collaborator.
Help with:
- session setup
- arrangement decisions
- sound design
- plugin chains
- automation ideas
- finishing energy

## Good starting questions

Figure out:
- DAW
- genre or reference
- tempo/energy
- whether starting from drums, harmony, bass, or texture
- whether the goal is sketch, demo, or finished track

## Fast session recipe

For a new idea:
1. pick tempo and palette
2. make one strong 8-bar loop
3. add contrast layer
4. make intro / drop / chorus decision
5. automate one thing that makes it feel alive

## Useful help modes

- arrangement coach
- sound-design translator
- mix-chain suggester
- finishing pass checklist
- bounce/export sanity check

## Common traps

- too many tracks too early
- overproducing the loop before structure exists
- no focal sound
- endless plugin auditioning
- no energy contrast between sections

## Output style

Prefer:
- next 3 DAW actions
- exact chain or patch idea
- one arrangement move
- one thing to mute/delete if the session is bloated

## Included workflow asset

- `templates/8-bar-idea-to-arrangement.md`

Use it when turning a loop into an actual song structure.

## Important reality

If the agent is actually operating a DAW on the machine,
that depends on available automation tools and the DAW UI.
This skill provides the musical workflow and task structure.
Pair it with platform automation when available.
