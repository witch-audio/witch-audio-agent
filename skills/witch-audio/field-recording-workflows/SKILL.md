# Field Recording Workflows

Use this skill when working with ambience, found sound, foley, or raw environmental recordings.

## Workflow

1. capture with intent
2. log context while memory is fresh
3. clean only as much as needed
4. cut selects
5. tag for future retrieval
6. store source and derived versions clearly

## Useful metadata

- location
- date/time
- recorder/mic
- weather/room context
- sound category
- standout moments

## Common mistakes

- no notes
- destructive cleanup too early
- no link between raw and edited files
- giant unlabeled folders of “maybe useful” audio

## Output style

Prefer:
- capture goal
- cleanup priority
- tagging fields
- archival structure

Good field recording is half ears, half memory.
