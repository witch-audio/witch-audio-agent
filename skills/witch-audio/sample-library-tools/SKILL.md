# Sample Library Tools

Use this skill when organizing sample packs, building sample tools, or cleaning huge audio libraries.

## Core problem

Most sample libraries fail in retrieval, not content.

## Useful operations

- filename normalization
- tag cleanup
- BPM/key extraction where possible
- one-shot vs loop separation
- preview generation
- duplicate detection
- folder/category cleanup
- provenance tracking

## Naming principles

Good names should reveal:
- source type
- character
- tempo/key if useful
- variation index

## Good output style

Prefer:
- folder strategy
- naming convention
- metadata fields
- what to automate first

The goal is searchable sound, not pretty folders.
