# Spatial Audio

Use this skill when the task involves width, placement, depth, binaural cues, or immersive sound design.

## Core dimensions

Think in:
- left/right
- front/back
- near/far
- dry/wet
- spectral cue
- timing cue

## Common mistakes

- widening that collapses in mono
- fake depth with only reverb
- too much movement with no anchor
- phase tricks that sound impressive alone and bad in context

## Fast checks

- mono fold-down
- phase/correlation
- headphones vs speakers
- static source vs moving source perception

## Output style

Prefer:
- spatial goal
- cue types to use
- mono safety check
- one practical implementation path
