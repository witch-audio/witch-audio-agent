# Ableton Live Workflows

Use this skill when the user is in Ableton Live.

## Best uses

- plugin QA sessions in Live
- rack/preset testing
- loop-to-arrangement workflow
- bounce and recall checks
- fast idea generation with templates

## Good Live harnesses

- audio-effect test set
- synth/polyphony stress set
- automation stress set
- save/reopen state recall set
- bounce comparison set

## Live-specific habits

- use labeled tracks and fixed device slots
- save tiny test sets, not giant production projects
- keep one “known clean” template
- test Session View and Arrangement View assumptions separately when needed

## For production

Great short workflow:
1. build one loop
2. identify the hook track
3. commit one automation move
4. duplicate into structure
5. bounce v1 fast

## Output style

Prefer:
- exact Live set to create or use
- exact track/device slot to inspect
- exact automation or playback move
- one clean next action
